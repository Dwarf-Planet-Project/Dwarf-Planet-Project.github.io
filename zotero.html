Zotero Server Installation Tool

Project Scope

The initial version focused on automating the installation steps provided by sualk in his github repository for the zotero data server. This development has stalled. New versions aim s on installing the zotero dataserver using the vanilla sources provided by the upstream project.

Roadmap

The development until now is focussed on the first alpha version, which will provide the basic functionality of the zotero data server.

Downloads

No tagged release was published until now. The development snapshot can be obtained from Git repository
